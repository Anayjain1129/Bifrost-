### What is this project? 
This is a Split Keyboard made by a 15 year old teenager(me) from india. it uses XIAO nRF52840 as MCU and a battery connection too! which uses ZMK in firmware! 

### How to use it?
1. Break the PCB in 2 halves! 
2. Assemble the PCB with soldering iron by soldering the MCU, switches, and diodes
3. Plug in the MCU with PCB and then put the firmware file in it
4. Drag and drop the firmware file in the MCU  
5. Do the same things with other XIAO 
6. That's now you have your split keyboard working
7. Make sure to put it in an enclosure

### Why did i made this? 
I made this project to learn how does the XIAO-nRF52840-SMD MCUs work and how does the battery work with the PCBs and also to learn how to solder the SMD components, as they are so small in size and to master the soldering of componentes by hand.

# Features
- Seeed Studio XIAO 
- Custom Layout
- ZMK firmware
- Very cool case/enclosure to hold the PCB
- Uses nRF52840
